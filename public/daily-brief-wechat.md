## 🔥 重磅 ChatGPT 图像生成量突破 15 亿次/周

OpenAI 公布了一个惊人数字：ChatGPT 每周图像生成量已经突破 **15 亿次**。按这个速度算，平均每秒要出图 2500 张，每天超过 2 亿张。自 Images 2.0 发布以来，用户的使用场景已经从"玩一玩"扩展到了设计草稿、产品原型、社交媒体素材等真实工作流。

**为什么值得关注？**
- **图像生成从小众玩具变成了大众日常工具。** 15 亿次/周意味着 AI 做图已经不是极客专属，普通人也在每天用。
- **数量背后是用例的爆发。** 不只是"画一张猫的照片"，而是产品设计草稿、电商主图、PPT 插图——这些能直接省钱的场景正在大规模落地。

## ⚡ 突破 谷歌搜索框全面升级

谷歌推出了基于 Gemini 3.5 的全新智能搜索框，把 AI Overviews 和 AI Mode 整合成了统一的搜索体验。最大的变化是支持**跨模态推理**——你可以直接丢一张图、一段视频、一个 PDF 文件进去，搜索框能读懂内容并给出答案，还能多轮对话追问。

**为什么值得关注？**
- **搜索框正在变成万能入口。** 以后不用切换 App，一个框搞定所有查询。
- **对小白用户最直接的改变：** "会搜索"正在变成"会问问题"。你不需要记各种搜索语法，直接用自然语言描述需求就行。

## 💡 趣味 Claude 官方确认：HTML 正在取代 Markdown

Claude 官方博客发文，解释为什么 Claude Code 团队正从 Markdown 转向 HTML 作为主要输出格式。原因是 HTML 能支持表格、CSS 样式、SVG 图表和 JavaScript 交互，信息密度和可读性远超纯文本。而且生成的 HTML 文件可以直接双击打开、直接分享给同事，不需要部署。

**为什么值得关注？**
- **昨天我们刚聊过"2 分钟生成 HTML 页面"，今天官方发博客确认这个趋势。**
- **对非程序员来说，HTML 正在变成"新的 Word 文档"。** 你不需要懂代码，只需要会说清楚需求，AI 就能帮你生成一份格式漂亮、能交互的文档。

[阅读原文 →](https://claude.com/blog/using-claude-code-the-unreasonable-effectiveness-of-html)
